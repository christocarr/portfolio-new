URL Shortener

coding challenge

Another coding challenge by Frontend Mentor.

Built using React and CSS modules and an API call to shrtco.de that returns a shortened url version of the url that the user entered.

What I learnt?

To learn more about testing I concentrated on a test driven development approach using Jest and Testing Library while building some components. I also learnt that by using Testing Library's priority query - getByRole. The developer then builds user interfaces with accessibility in mind because they are then forced to code elements that will be exposed on the accessibility tree on the DOM.
